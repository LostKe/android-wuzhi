# APP已实现的功能

* 此刻
* 写新日记
* 查看历史日记
* 修改昵称，签名
* 修改隐私设置

# 关于
* 开发该App 纯属兴趣爱好。如发现bug 请提issues,或者pull requests。谢谢！
